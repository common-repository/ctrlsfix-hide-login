== Description ==
The easy way to hide wp-login.php file.
You can easily redirect url with slug url .
Demo : `http://ctrlsfix.com?slug_text`
Go to `Settings >Hide Login` .

= Features =
* Block default wp-login
* Providing security

== Frequently Asked Questions ==

Go to `Settings > Hide Login` and add slug text.
= What is default slug text ? =
Your default slug text is `root`
Demo : `ctrlsfix.com?root`
